CMSIS NN Lib example arm_nnexample_cifar10 for
  Cortex-M4 and Cortex-M7.

The example is configured for uVision Simulator as well as the STM32F407 DISCOVERY board.
For more information, please read the following tutorial
[How to run deep learning model on microcontroller with CMSIS-NN](https://www.dlology.com/blog/how-to-run-deep-learning-model-on-microcontroller-with-cmsis-nn/)
